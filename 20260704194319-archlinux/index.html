    关于arch linux的安装教程#

    • 使用 setfont-132tar 设置终端大号字体，使用 ip link 检查网卡，通过iwctl工具进行无线网连接。
    • 通过 ping (website) 检查网络连接是否成功。
    • 随后使用fdisk工具对磁盘进行分区, 并格式化format分区，挂载mount分区。
    • 更换本地源origin可以提升下载速度降低失败率。
    • 现在可以使用pacstrap安装必要的固件。
    • 设置setting系统开机自动挂载规则并进入系统。
    • 设置setting时区并写入硬件。
    • 启动enable必要的服务。
    • 主机本地化local。
    • 设置管理员密码password，创建用户。
    • 增强字体font渲染。
    • 安装引导grub程序。
    • 最后通过 umount -R /mnt 卸载挂载点。
